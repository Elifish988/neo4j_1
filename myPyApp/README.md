# מבנה עץ התיקיות של my_flask_app

```python
myPyApp
│
├── app.py           # הקובץ הראשי של האפליקציה
├── config.py        # קובץ קונפיגורציה (למשל, חיבור לבסיס נתונים)
│
├─┬ models           # תיקיית מודלים
│ ├── init.py        # קובץ ריק, עושה את המודל לתיקיית מודלים
│ └── user.py        # מודל עבור משתמשים
│
├─┬ routes           # תיקיית ראוטים
│ ├── init.py        # קובץ ריק, עושה את הראוטים לתיקיית ראוטים
│ └── user_routes.py # קובץ ראוטים עבור משתמשים
│
├─┬ templates        # תיקיית תבניות HTML
│ ├── index.html     # תבנית לדף הבית
│ └── users.html     # תבנית להציג את המשתמשי
│
├── static           # תיקיית קבצים סטטיים (CSS, JavaScript, תמונות)
└── style.css        # קובץ CSS לעיצוב
```
